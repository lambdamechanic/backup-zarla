¿Cómo elijo la tipografía para el logotipo de una revista?

Dependiendo de tu público objetivo y del tipo de contenido que vayas a publicar, puedes elegir entre las diversas tipografías. Parisienne y Crushed serán ideales para revistas femeninas, mientras que Marko One y Fredoka One funcionarán bien para revistas infantiles. Las publicaciones comerciales serias se beneficiarían de estilos como Bebas Neue u Oswald.
